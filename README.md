# JavaScript Learning Journey 🚀

This repository is my personal collection of notes, code exercises, and mini-projects as I advance my skills in JavaScript and Node.js.

## 📂 Repository Structure

I have organized the content by topic for better tracking:

- `basics/`: Fundamental JS concepts (variables, loops, functions, scopes).
- `es6-plus/`: Modern ES6+ features (Arrow Functions, Destructuring, Promises, Async/Await).
- `dom-manipulation/`: Browser-based experiments and UI interactions.
- `node-exercises/`: Backend development practice (Node.js v22).
- `challenges/`: Algorithmic problems and logic puzzles.

## 🛠 Tech Stack

- **Language:** JavaScript (ES6+)
- **Runtime:** Node.js v22
- **Editor:** VS Code

## 🚀 Getting Started

To run any of the scripts in this repository, navigate to the specific folder and run the file via your terminal:

```bash
cd basics
node index.js
```
